# sudoku-project
AP CSA Sudoku Project
Brandon Sweet

This code generates a valid sudoku board with numbers one to nine that don't overlap with any other number in the same box row or column. To run the program, paste the code into coding space and press run. The sudoku board is generated by recursively changing one digit of the board to a random number one through nine, and verifying that the board so far as a valid solution. If the code ever runs into a space where none of the numbers one through nine are our valid solutions, it backtracks and resets the last however many numbers until it can find a valid solution. It uses a 4D array, where the first two dimensions represents each of the 3 x 3 boxes, and the last two dimensions are the numbers within each of those boxes. There are two files included in the repo; sudoku.java is the file with the code, and there is a text file containing a description of the project.
